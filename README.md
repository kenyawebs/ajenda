# Ajenda - Cultural Events Beyond Boundaries

A revolutionary global events platform born in Kenya, connecting cultures through authentic event experiences across all 27 event categories.

## Features

- **27 Event Categories** - Complete coverage from Arts & Entertainment to Government Functions
- **3D Interactive Globe** - Explore events on an animated 3D globe with particle effects
- **170+ Event Sub-types** - The most comprehensive event classification system
- **AI Cultural Intelligence** - Smart recommendations based on cultural context
- **Immersive Design** - Futuristic UI with glass morphism, particle systems, and GSAP animations
- **Kenya Priority Index** - Categories ranked for Kenyan market relevance
- **Global Coverage** - 120+ countries and growing

## Tech Stack

- React 19 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Three.js / React Three Fiber (3D globe & particles)
- GSAP + ScrollTrigger (scroll animations)
- Framer Motion (component animations)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
